# Custom Sprites
Custom actors/sprites I've made.

Here's what's currently available:
- SMM2-like 10/100 Coin (remake of surrealismtheta's)

## Setup
Define the profile name for the sprite(s) in `actor.h`, `profileid.h`, and `asm_profiles.S`.
Add the YAML files to your project file (e.g. `NewerProject.yaml`)
